2. Approximability
3. Abstraction and composability
4. Exploration

Repeat these 4 pillars to form a compounding positive feedback loop.

## MVP ideas

Here is a mathematical formulation of a strawman MVP:

- **Input:** Non-random Stream (time series) of numbers (tokens) generated by some possibly complicated mathematical function. e.g. Quadratic function or even Mandelbrot fractal.
- **Goal:** Predict next N number(s) with highest accuracy, i.e. minimal delta between actual and predicted
- Let's only consider abstraction and composition pillar for now. Feedback loop and approximation are already given.
- **Next level of complexity:** Consider an output token as well generated by the AI which influences the next token.

For composition, we start with the set of basic functions:

- e.g. linear, quadratic, polynomial etc.
- trigonometric
- combinations of any of the above, i.e. composition, addition, subtraction, multiplication, division, etc.
  - Note: composition means things like g(f(x)), e.g. sin(x**2 + log(x) + 3)

Understand the power of this technique. Instead of some combination of neurons, the AI is going to keep candidates for what might be the possible generating function, using a composition of known types of functions. Then it is going to also try to guess any coefficients.

If we guess well, then the size of the AI will be very small, and it will also be very accurate.
